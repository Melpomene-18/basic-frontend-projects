Common JavaScript Functions

Jul 1, 2026 · JavaScript

JavaScript adds interactivity to web pages. Below are everyday functions and patterns you will see constantly.

Querying the DOM

Select HTML elements to read or modify them.

// Single element
const header = document.querySelector("h1");

// All matching elements
const items = document.querySelectorAll(".item");

// Legacy alternatives
document.getElementById("logo");
document.getElementsByClassName("card");

Modifying Elements

Change text, HTML, classes, and attributes after the page loads.

const el = document.querySelector(".message");

el.textContent = "Hello, world!";
el.classList.add("visible");
el.setAttribute("data-id", "42");
el.style.color = "red";

Event Listeners

Respond to clicks, key presses, form submissions, and more.

button.addEventListener("click", () => {
    console.log("Button was clicked");
});

form.addEventListener("submit", (event) => {
    event.preventDefault();
    console.log("Form submitted");
});

Array Methods

Modern JavaScript leans heavily on array iteration.

const numbers = [1, 2, 3, 4, 5];

const doubled = numbers.map(n => n * 2);
const evens = numbers.filter(n => n % 2 === 0);
const sum = numbers.reduce((acc, n) => acc + n, 0);

numbers.forEach(n => console.log(n));

Working with Objects

Objects store key-value pairs. Access properties with dot or bracket notation.

const user = { name: "Alice", age: 30 };

console.log(user.name);
console.log(user["age"]);

const keys = Object.keys(user);
const values = Object.values(user);
const entries = Object.entries(user);

Fetch API

Make HTTP requests to load data from a server.

async function getData() {
    const response = await fetch("/api/posts");
    const data = await response.json();
    return data;
}

getData().then(posts => console.log(posts));

Template Literals

Embed variables and expressions directly inside strings.

const name = "Bob";
const greeting = `Hello, ${name}! You have ${3 + 2} notifications.`;

Default Parameters and Arrow Functions

Arrow functions are concise. Default parameters provide fallback values.

const greet = (name = "friend") => `Hi, ${name}!`;

const add = (a, b) => a + b;

Local Storage

Persist small bits of data in the browser.

localStorage.setItem("theme", "dark");
const theme = localStorage.getItem("theme");
localStorage.removeItem("theme");

Most Common CSS Commands

Jun 24, 2026 · CSS

CSS (Cascading Style Sheets) controls the visual presentation of a web page. Here are the properties you will reach for most often.

Color and Background

Set text and background colors. Use hex, rgb(), or named values.

body {
    color: #333;
    background-color: #f9f9f9;
}

Font Styling

Control typeface, size, weight, and line height.

p {
    font-family: "Helvetica", sans-serif;
    font-size: 1rem;
    font-weight: 400;
    line-height: 1.6;
}

The Box Model

Every element is a box. margin pushes outward, padding pushes inward, and border sits between them.

.card {
    width: 300px;
    margin: 1rem auto;
    padding: 1.5rem;
    border: 1px solid #ddd;
    border-radius: 8px;
}

Display

display determines how an element flows in the layout. Common values are block, inline, flex, and grid.

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 1rem;
}

Position

relative shifts an element from its normal spot. absolute removes it from the flow and positions it relative to the nearest positioned ancestor. fixed pins it to the viewport.

.tooltip {
    position: absolute;
    top: -0.5rem;
    right: 0;
}

Flexbox

Flexbox aligns items along one axis. Ideal for navigation bars, card rows, and centering.

nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem 2rem;
}

Grid

CSS Grid lays out items in two dimensions. Define columns and rows, then place children.

.gallery {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 1rem;
}

Transitions and Hover

Animate property changes smoothly with transition. Combine with pseudo-classes like :hover.

button {
    background-color: blue;
    transition: background-color 0.2s ease;
}

button:hover {
    background-color: darkblue;
}

Responsive Design

Use @media queries to adapt layouts to different screen sizes.

@media (max-width: 600px) {
    .gallery {
        grid-template-columns: 1fr;
    }
}

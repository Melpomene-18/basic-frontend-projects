Basic HTML Commands and Their Usages

May 1, 2026 · HTML

HTML (HyperText Markup Language) is the backbone of every web page. Below are the most essential elements you will use when writing HTML.

Document Structure

Every HTML document starts with a standard skeleton that tells the browser how to interpret it:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <!-- content here -->
</body>
</html>

Headings

Headings range from <h1> down to <h6>. Use <h1> once per page for the main title, then nest subheadings in order.

<h1>Main Title</h1>
<h2>Section</h2>
<h3>Subsection</h3>

Paragraphs and Text

The <p> tag wraps a block of text. For inline emphasis use <strong> for bold and <em> for italic.

<p>A paragraph of <strong>bold</strong> and <em>italic</em> text.</p>

Links

The anchor tag <a> creates hyperlinks. The href attribute sets the destination URL.

<a href="/about.html">About page</a>

Images

The <img> tag is self-closing. Always include an alt attribute for accessibility.

<img src="photo.jpg" alt="A scenic mountain view">

Lists

Unordered lists use bullet points; ordered lists use numbers.

<ul>
    <li>Item one</li>
    <li>Item two</li>
</ul>

<ol>
    <li>First step</li>
    <li>Second step</li>
</ol>

Forms

The <form> element collects user input and sends it to a server.

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <button type="submit">Send</button>
</form>

Divisions and Spans

<div> is a block-level container; <span> is inline. Both are used for grouping and styling.

<div class="card">
    <span class="badge">New</span>
</div>

Semantic HTML

Semantic tags like <header>, <nav>, <main>, <section>, and <footer> describe the purpose of content blocks. They improve accessibility and SEO.

<header>
    <nav>...</nav>
</header>
<main>
    <section>
        <h2>Introduction</h2>
    </section>
</main>
<footer>...</footer>
